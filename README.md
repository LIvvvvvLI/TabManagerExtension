# VSCode Tab Manager - 快速开始
`This is a VS Code extension developed based on GLM-5 (all code is AI-generated).`

## 这是什么？

这是一个 VSCode 扩展，提供类似 IDEA 的标签页管理功能。

## 主要功能

1. **显示所有标签页列表** - 在一个可滚动的列表中查看所有打开的标签页
2. **批量关闭标签页** - 多选标签页后一次性关闭
3. **快捷关闭操作** - 关闭左侧/右侧所有标签页
4. **编辑器按钮** - 在编辑器标题栏添加快捷按钮

## 快速开始

### 安装依赖
```bash
npm install
```

### 编译
```bash
npm run compile
```

### 调试
1. 在 VSCode 中打开此项目
2. 按 F5 启动调试
3. 在新窗口中测试扩展

### 打包
```bash
vsce package
```

## 使用扩展

1. 打开多个文件
2. 按 `Ctrl+Alt+T` 或点击编辑器标题栏的按钮
3. 在弹出的列表中选择要关闭的标签页
4. 按 Enter 关闭选中的标签页
